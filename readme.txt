== Description ==
Implements the "rating" custom field for use with oik-fields and custom post types

== Installation ==
1. Upload the contents of the oik-rating plugin to the `/wp-content/plugins/oik-rating' directory
1. Activate the oik-rating plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==
= How does it work? =
Uses the jQuery raty script to display a 5 star rating field.
